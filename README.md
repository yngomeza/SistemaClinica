# Sistema de gestión de Clínica Médica

## Descripción
Proyecto final desarrollado en C++ para la administración de una clínica médica. El sistema permite registrar pacientes, programar citas, gestionar el historial de consultas y generar reportes diarios.

## Funcionalidades
- Registro de pacientes.
- Programación de citas médicas.
- Historial de consultas.
- Reportes diarios.
- Búsqueda y ordenamiento de información.

## Tecnologías
- Lenguaje: C++
- Estructuras
- Archivos
- Búsqueda
- Ordenamiento
